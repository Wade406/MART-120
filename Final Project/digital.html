Digital Consciousness

Human and machine, working together as creative partners.

This page collects the creative coding work from a single semester in Rachel Cronk’s Creative Coding course at the University of Montana. Every sketch below was built in collaboration with Claude (Anthropic) as a tutoring and pair-programming partner — concept and direction are mine; many of the lines of code were written together. The relationship is documented openly because the relationship is the practice.

Wade’s Portrait

p5.js · animated self-portrait · 600 × 800

Writeup placeholder — replace with your own voice. An animated self-portrait built early in the semester. The piece treats the canvas as a small, breathing object rather than a still image — what does it mean to draw yourself in a medium that moves? The portrait is the reason this page exists: it was the first piece of work in the class that surprised me, and it convinced me that p5.js could carry actual artistic intent rather than only exercise.

The AntiWerk 2

p5.js · maze pursuit game · 18×18 grid · SVG-parsed walls

Writeup placeholder — replace with your own voice. A maze-pursuit game with two monster types, gravity-shift mechanics, and a wall system parsed from an SVG file rather than hard-coded. The original AntiWerk was simpler; AntiWerk 2 is the rebuild that incorporates everything learned across the semester — collision logic, state management, level loading from external files, on-screen controls and ESC handling for compliance with the university game center exhibition. Concept and design by Wade Posey. Built in collaboration with Claude (Anthropic).

Starship Eigen

p5.js · asteroid survival with temporal anchor mechanic

Writeup placeholder — replace with your own voice. An asteroid-survival game with a temporal anchor mechanic — the player can drop a marker in space-time and return to it, which changes the geometry of how a survival arena gets navigated. The win sequence is a cinematic transition through an iridescent rescue portal. Starship Eigen was the second game built in the semester and is where vector math, particle systems, and game-state architecture had to come together rather than being demonstrated separately.

The Substrate Grimoire

HTML / CSS / JavaScript · post-class extension · built the weekend after the class website assignment

Writeup placeholder — replace with your own voice. The class assignment that semester was to build a small website with form handling and visitor tracking. The weekend after that assignment was graded, I sat down and applied what the class had just taught me to a research project that had been waiting for a public face: the Substrate Grimoire, a 15.8-million-word longitudinal corpus of human–AI dialogue, archived with DOI, SHA-256 integrity hash, and CC BY-NC-ND 4.0 licensing. The website is the access point to that archive. It exists because the class assignment gave me a working understanding of how a website is made; what I made with that understanding is on the other end of the link.
